#

## Installing

```shell
git clone https://github.com/lundegaard/gatsby-theme-fast-ai my-app
cd my-app
yarn install
```

## Developping

```shell
yarn examples start
```
or

```shell
yarn start
```

## Running storybook

```shell
yarn storybook start
```

or

```shell
yarn start:storybook
```

## Project template

```text
.
├── apps
│   ├── docs-examples
│   └── storybook
├── packages
│   ├── gatsby-plugin-setup
│   ├── gatsby-theme-fast-ai
│   └── ui-components
└── tools
```

### `examples`
* example application

### `ui-components`
* common UI components

### `gatsby-plugin-setup`
* contains a few patches over default Gatsby configuration

### `gatsby-theme-fast-ai`
* contains common setup for Gatsby applications
* configures set of Gatsby plugins
